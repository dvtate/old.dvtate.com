Current Favorite Language:

    Erlang is a functional programming language, it has 'variables', but you can't change their value. Erlang is popular because it is a concurrent language so, it supports multi-threading and can use more than one core/processor, which means it can do more than one thing at a time, making it very fast.

I attempted to send it via email, but Gmail wouldn't let me send it. So I converted it to HTA, ASP, and finally PHP (I had never heard of JavaScript). This was my first addition to this website.
